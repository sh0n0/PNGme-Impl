# PNGme-Impl
My Implementation of [PNGme](https://picklenerd.github.io/pngme_book/).

## Usage

### Encode
```
$ pngme encode ./image.png ruSt "This is a secret message"
```

### Decode
```
$ pngme decode ./image.png ruSt
```

### Remove
```
$ pngme remove ./image.png ruSt
```

### Print
```
$ pngme print ./image.png 
```
